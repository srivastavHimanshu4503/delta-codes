Welcome to Random Number Generator

This webpage is used to play the game "Guess the number" in 10 chances.

Rules to play the game:

  • User needs to guess the number between his choice i.e, he has the power to choose the range for the game.
  • If it is correct congrats you won else it tells you to either increase the number or decrease it.
  • User needs to guess the number before his/her all chances expires.
  • If he/she is unable to guess the number in the available chances he/she left, then user will lose the game.
